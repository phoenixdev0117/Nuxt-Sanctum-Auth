## v0.1.0

- Initial release
- Automated CSRF token management
- Bearer token management
- Supports both CSR and SSR
- Compatible with Nuxt's `ofetch` client
- `useSanctumAuth` composable
- **Authentication:**
    - Login
    - Logout
    - Get current user
    - Refresh User
